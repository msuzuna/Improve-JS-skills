## 開発環境
### 初期設定
nodeバージョン：18.16.0

コマンド
```node
yarn install
```

### 開発
watchタスクとbuildタスクがあります。
```node
yarn run watch
yarn run build
```
サーバーの立ち上げはありませんので、VScodeのLive Server等を活用ください。
